Description: This is lab 3, simple notekeeper. This application when run, will show
viewnote page .This page let user to read title and contents. 
To edit text in the above columns, there is a link present "Edit" on the viewnote page.
When user hits the link, it will bring editnote page which has textarea allowing user to 
enter title and contents. After filling out title and content, a save button is added
 to save the changes done by the user. After clicking the save button, viewnote page 
will pop up , to show the changes to the user. Behind the scenes,there is a file name 
note.txt, which will save the input entered by the user.
